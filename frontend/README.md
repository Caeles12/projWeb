# FR-Administration : Frontend

Frontend de l'application FR-Administration, développé en Angular.

## :bricks: Build from source

### Dépendances

- Node.js

### Installation
```
bash
git clone
cd wm-projet
npm install
```

### Développement

```
bash
ng serve
```

Aller sur `http://localhost:4200/`.

```
ng test
ng e2e
```
